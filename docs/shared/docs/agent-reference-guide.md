## Data Sources & References

### Liquidity Regime Data Sources
- **Fed Balance Sheet:** FRED (WALCL)
- **Treasury General Account:** Treasury Direct/FRED
- **Reverse Repo Facility:** FRED
- **Real Yields:** FRED (10-year TIPS yield)
- **Options Market:** Gamma/skew indicators
- **BTC ETF Flows:** Farside Investors, ETF providers
- **Stablecoin Supply:** CoinGecko/DeFiLlama
- **BTC Dominance:** CoinGecko/CoinMarketCap
- **Funding Rates:** Major exchanges (perpetual futures)
- **Open Interest:** Futures OI trends
- **Exchange Flows:** Glassnode, CryptoQuant
- **On-Chain Metrics:** HODL waves, active addresses, hash rate

### Performance Benchmarking
- S&P 500
- BTC
- Sector/asset class performance data

## Update Cadences

- **Daily:** No active monitoring required
- **Weekly:** Data processing, liquidity regime monitoring (information only)
- **Quarterly:** Full portfolio review, rebalancing, strategy updates
- **Annually:** Full portfolio audit, real estate review, succession/inheritance update

---

## Exception Protocol

Exceptions to rules are **only** allowed if:
1. Life-threatening emergency requiring immediate liquidity
2. Legal/regulatory requirement that cannot be deferred
3. Written exception request approved in quarterly review

**Default:** No exceptions. Rules are designed to be followed without exception.
