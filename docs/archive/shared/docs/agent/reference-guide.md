# Agent Reference Guide

**Purpose:** Key reference documents, data sources, update cadences, exception protocol.

**Strategy:** `/strategy/strategy/financial-strategy.md` (framework), `/strategy/strategy/financial-goals.md` (goals), `/strategy/strategy/financial-background.md` (constraints), `/strategy/strategy/psychological-profile.md` (behavioral mandates)

**Tactics:** `/strategy/tactics/financial-tactics.md`, `/strategy/tactics/liquidity-centered-rebalancing-framework.md` (three-axis rebalancing), `/strategy/tactics/stx-playbook.md` (STX management)

**Operations:** `/strategy/operations/operating-manual.md` (daily/weekly/quarterly/annual rules), `/strategy/operations/finance/quarterly-portfolio-review-process.md` (workflow), `/strategy/operations/finance/quarterly-portfolio-review-template.md` (template), `/strategy/operations/finance/quarterly-rebalancing-checklist.md` (checklist), `/strategy/operations/finance/transaction-import-process.md`, `/strategy/operations/finance/data-import-process.md`

**Reference:** `/strategy/reference/btc-liquidity-regime-scorecard.md`, `/strategy/reference/altcoin-liquidity-regime-scorecard.md`, `/strategy/reference/liquidity-regime-model.md`, `/strategy/reference/scorecard-efficacy-assessment.md`

**Data Sources:** FRED (Fed Balance Sheet/WALCL, Treasury General Account, Reverse Repo, Real Yields), Farside Investors/ETF providers (BTC ETF flows), CoinGecko/DeFiLlama (Stablecoin supply, BTC dominance), Exchanges (Funding rates, Open Interest), Glassnode/CryptoQuant (Exchange flows, on-chain metrics)

**Cadences:** Daily: none. Weekly: data processing, liquidity monitoring (info only). Quarterly: portfolio review, rebalancing, strategy updates. Annually: portfolio audit, real estate review, succession update.

**Exceptions:** Only for life-threatening emergency, legal/regulatory requirement, or written exception approved in quarterly review. Default: no exceptions.

**Workflows:** `/strategy/operations/README.md` (complete docs). Quarterly: `/strategy/operations/finance/quarterly-portfolio-review-process.md`. Agent reqs: `/shared/docs/agent/rules/workflow-requirements.md`.
