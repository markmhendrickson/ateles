# Agent Decision-Making Framework

**Purpose:** Framework for decisions, document creation/updates, process execution while maintaining consistency.

**Making Recommendations:** Check strategy (`/strategy/strategy/`), review tactics (`/strategy/tactics/`), follow operations (`/strategy/operations/[domain]/`), respect behavioral mandates (`/strategy/operations/operating-manual.md`), use established frameworks.

**Creating/Updating Documents:** Maintain hierarchy (Strategy > Tactics > Operations), preserve consistency, document decisions, save outputs (`/shared/docs/agent/rules/workflow-requirements.md`), follow naming (YYYY-MM-DD format).

**Executing Processes:** Follow established workflows (`/strategy/operations/[domain]/`), save results (mandatory for finance), verify compliance (`/strategy/operations/operating-manual.md`), update documents.

**Behavioral Compliance:** Verify compliance with mandates. Quarterly reviews verify adherence. Respect domain constraints (finance: no discretionary trading, quarterly-only cadence). No violations of forbidden actions. Trading: quarterly-only. Monitoring: no daily portfolio monitoring. Concentration: see `/strategy/tactics/stx-playbook.md` for STX limits.

**Reference:** `/strategy/operations/operating-manual.md` for complete behavioral rules and constraints.
